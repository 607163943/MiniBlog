# 接口设计

后端接口基础url：http://localhost:8080/api

### 管理端

* 工作台
  * 统计卡片区域：GET /dashboard/stats
  * 专栏文章分布（饼图 + 表格）：GET /dashboard/article-distribution
  * 文章发布日历热力图：GET /dashboard/article-calendar
  
* 专栏模块

  * CRUD请求路径：/admin/blog_column
* 文章模块
  * CRUD请求路径：/admin/article
* 标签模块
  * CRUD请求路径：/admin/tag
