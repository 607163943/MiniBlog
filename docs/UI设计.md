* UI配色方案

  * ### 1. 顶栏（Header）

    - 背景色：`#FFFFFF`（纯白，简洁清爽）
    - 文字/图标：`#303133`（深灰，保证对比度）
    - Hover/选中：`#409EFF`（呼应 Logo 主色）
    - 分割线：`#E4E7ED`（浅灰，弱化边界）

    👉 这样 Logo 颜色会和顶栏形成呼应，而不会“掉进去”。

    ------

    ### 2. 侧边栏（Sider / Menu）

    - 背景色：`#F5F7FA`（浅灰，避免和主内容完全融合）
    - 菜单文字：
      - 默认：`#606266`（中灰）
      - 选中：`#409EFF`
    - Hover 背景：`#E6F0FF`（蓝色浅底，柔和点缀）

    ------

    ### 3. 主内容区域（Content）

    - 背景色：`#FAFAFA`
    - 卡片背景：`#FFFFFF`
    - 卡片边框：`#E6E8EF`（淡灰边框）
    - 标题文字：`#303133`
    - 正文文字 `#4B5563`
    - 次级文字：`#606266`
    
    ------
    
    ### 4. 工作台卡片（Dashboard Stats）
    
    保持色彩活泼，用轻快的四个主色区分不同统计数据：
    
    - 文章总数 → 蓝色 `#409EFF`
    - 专栏总数 → 青绿色 `#17C3B2`
    - 标签总数 → 橙色 `#FF9F1C`
    - 用户数 → 紫色 `#9D4EDD`
    
    👉 颜色更「年轻、清新」，避免太多红绿搭配。
    
    ------
    
    ### 5. 表格（文章/专栏/标签管理）
    
    - 表头背景：`#FAFAFA`
    - 表头文字：`#303133`
    - 表格行：交替 `#FFFFFF` / `#FAFAFA`
    - 状态：
      - 草稿 → 灰色 `#909399`
      - 发布 → 蓝色 `#409EFF`
      - 下架 → 橙色 `#FF9F1C`
    - 行悬停浅蓝 `#EDF2FF`
    - 选中浅蓝 `#E6F0FF`
    - 危险动作：`#EF4444`

    ------
    
    ### 6. 按钮

    - 主按钮：背景 `#409EFF`，文字 `#FFFFFF`
    - 次按钮：背景 `#FFFFFF`，边框 `#409EFF`，文字 `#409EFF`
    - 危险按钮：背景 `#F56C6C`，文字 `#FFFFFF`

    ------
    
    ## 7. 分页组件（Pagination）
    
    - 背景色：透明（随内容区背景）
    - 页码文字：
      - 默认：`#606266`（中灰）
      - 选中：`#409EFF`（主色）
      - 禁用：`#C0C4CC`（淡灰）
    - Hover：`#409EFF`（蓝色描边或文字高亮）
    - 边框：`#EBEEF5`（淡灰，保持和表格边框一致）

    👉 这样分页条会轻盈、不突兀，但仍然和表格形成呼应。

    ------
    
    ## 8. 对话框组件（Dialog）
    
    - 标题区：
      - 背景色：`#F5F7FA`（浅灰）
      - 标题文字：`#303133`
      - 关闭按钮 Hover：`#F56C6C`（红色提示）
    - 内容区：
      - 背景色：`#FFFFFF`
      - 文字：`#303133` 主文字，`#606266` 辅助文字
    - 底部按钮区：
      - 边框：`#EBEEF5`（上边分割线）
      - 按钮沿用全局按钮方案：
        - 主按钮 → 蓝色 `#409EFF`
        - 次按钮 → 白底蓝字
        - 危险按钮 → 红色 `#F56C6C`
